# Timeline 密度、置信度优先与研究影响门禁

本规格约束事件脉络在事件规模持续增长后的展示密度、研究质量和加载边界。

- [PRD.md](./PRD.md)：产品目标与验收口径
- [SYSTEM.md](./SYSTEM.md)：排序、研究影响门禁、月聚合和懒加载设计
- [UPSTREAM-RESEARCH.md](./UPSTREAM-RESEARCH.md)：学术、同行认可与产业采用信源
- [TEST.md](./TEST.md)：契约、集成与浏览器验收
- [TASKS.md](./TASKS.md)：实施状态

核心原则：Timeline 仍只展示公开 Event，但“已公开”不等于“必须在默认视图占据同等注意力”。论文存在、内容完整和研究影响是三件不同的事；月份允许没有论文，只有能证明行业影响、领域影响或强早期采用的研究才进入 Timeline。低价值论文继续保留永久事件页与证据，不挤占主阅读路径。
